# aerc — Global

Ctrl + t                 Nouveau terminal
Ctrl + c / Ctrl + q      Quitter (confirmation)
Ctrl + z                 Suspendre
?                        Aide raccourcis

# aerc — Liste des messages

g / G                    Premier / Dernier message
Entrée                   Ouvrir message
q                        Quitter
