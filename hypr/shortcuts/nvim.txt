# LazyVim — Navigation de base
h / j / k / l           Gauche / Bas / Haut / Droite
w / b / e               Mot suivant / précédent / fin
gg / G                  Début / Fin du fichier
0 / ^ / $               Début ligne / Premier char / Fin ligne
{ / }                   Paragraphe précédent / suivant
Ctrl + F / B            Scroll avant / arrière
H / L                   Buffer précédent / suivant
s                       Flash (saut rapide)
S                       Flash Treesitter
%                       Aller au bracket correspondant

# LazyVim — Modes
i / a / o / O           Insérer avant / après / dessous / dessus
v / V / Ctrl + V        Sélection char / ligne / bloc
Ctrl + Space            Sélection Treesitter incrémentale
Escape                  Mode normal

. (point)               Répéter dernière action

ci( / di(               Changer / Supprimer dans parenthèses
